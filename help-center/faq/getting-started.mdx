## Quick Steps

1. Sign in at `looly.io`
2. Click the "New Project" button on your dashboard
3. Enter a name and optional description
4. Click "Create"

## Project Setup

Your project is ready! You can now add documentation pages, invite team members, and set project visibility.

<Callout kind="info">
Looly auto-saves your work as you edit documentation.
</Callout>
