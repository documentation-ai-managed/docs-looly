## What is Looly

Looly provides your default documentation space. You organize and manage all project documentation in one centralized location. Build structured guides, API references, and changelogs with ease using MDX-powered pages.

<Callout kind="info">
Looly supports version control, search, and collaboration features out of the box.
</Callout>

## Get Started Quickly

Follow these steps to launch your first documentation space.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com` and create your free account.
  </Step>
  <Step title="Create Space" icon="plus">
    Click "New Space" and name it for your project, like "MyApp Docs".
  </Step>
  <Step title="Add Content" icon="edit-3">
    Use the editor to create your first page. Start with Markdown or import from GitHub.
  </Step>
  <Step title="Publish" icon="globe">
    Deploy instantly and share your custom domain or subdomain.
  </Step>
</Steps>

<CodeGroup tabs="CLI,Direct">
  ```bash
  npx create-looly@latest my-docs
  cd my-docs
  npm run dev
  ```
  ```bash
  # Or use the dashboard import
  # Upload your existing Markdown files directly
  ```
</CodeGroup>
